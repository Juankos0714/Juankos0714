### 🌟 Featured Project

#### 🏨 [Ubik](https://github.com/juankos0714) — Hotel & Motel Reservation Platform

A full-stack reactive platform for motel and hotel management, deployed on Azure.

- **Architecture:** 7 reactive microservices (Spring WebFlux) — Gateway, User Management, Motel Management, Notifications, Payments, Streaks, AI Service
- **Frontend:** Angular with SSR, signal-based state, glassmorphism design system, light/dark theming
- **Infra:** Docker Compose · Azure VM · Nginx + SSL (Let's Encrypt) · systemd auto-startup
- **Integrations:** Stripe (payments + webhooks) · RabbitMQ (async events) · Ollama (local LLM) · Prometheus + Grafana + Loki (observability)
- **Patterns:** Hexagonal architecture · Reactive programming · Event-driven communication · Role-based auth (USER / OWNER / ADMIN)
